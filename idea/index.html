# 2022年6月11日

关注到唐山烧烤店围殴女性事件，针对现在网上的言论聊聊自己的看法。先表明态度：作为一名男性，也作为曾经校园暴力事件的亲历者，坚决支持依法严惩，而且本次事件应该已经算得上法律意义上的社会影响极其恶劣，不出意外应该会顶格处罚。

但同时，我也关注到一些“相当低智”的声音，用脚趾头都能想到是收钱办事的狗。它们以女子态度恶劣，言语挑衅为据加以指责，说她活该被打。先不说别的，希望它或者它家人被流氓性骚扰时，还可以心平气和地跟流氓讲道理。哦对，说不定没有家人。

本来和陌生女性搭讪，就要做好被拒绝，甚至被人调侃的准备。我估计事件主犯是因为被女性拒绝，觉得在兄弟面前丢了面子，想通过打人泄愤，找回他那脆弱的自尊心。成年人就该为自己的冲动负责，处理好受害者的赔偿事宜，老老实实进去踩缝纫机。

近些年，国内治安的确肉眼可见的越来越好。在我上高中之前，社会治安那叫一个乱，拦路抢劫的，入室盗窃的屡见不鲜。学校里也是三天两头打架斗殴，保安都不一定敢管。

我第一次遭遇校园暴力还是三年级，一名六年级的学生，拿着一根铝管来抢我的游戏王卡片，见我不答应，就拿管子往我身上抽。那时候我在同龄人里算比较壮实的，胆子也大，直接一把抓住管子抢了过来，对着它就是一顿猛砸，最后管子还被我拿走了。

体格占优势的情况下确实可以反击，但是大多数女性的体格是比不过男性的。所以一旦卷入打斗，最好的办法就是跑，然后报警。同时，尽量不要去言语激怒犯罪分子，先保全自己才是明智的选择。

暴力必须关进笼子里，施暴者必须严惩，但是希望各位保持理智，不要被网上某些别有用心的小人利用。这件事的本质是善与恶，正与邪的对抗，是全社会都要抵制和抨击的不良风气。

不是男女对立！不是男女对立！不是男女对立！

男性普遍是支持严肃处理的，对女性施暴的毕竟是个别，媒体要热度所以对这类事件大肆渲染，导致很多平和善良的男性无辜躺枪。不能让这一小撮人代表了全社会的男性，我们窝里斗，人家可是在屏幕前笑得合不拢嘴。

# 2022年5月24日

关于长沙LV专柜售假事件，这里面：品牌方、仓储、门店、店员、消费者，都是独立的，且拥有各自利益的个体。我不认为LV作为一个全球奢侈品品牌，会蠢到去卖假货。奢侈品能以市场价卖出去，本身就保证了高额的利润，卖假无异于杀鸡取卵。

在上述五个个体中，品牌方是最有可能说真话的，因为售假对它而言百害无一利，所以剩下的可能：

其一，仓储或者门店监守自盗，勾结制、售假产业链，假货上架到门店，真货卖到黑、灰色渠道。

其二，门店店员心生歹念，将消费者购买的真货掉包，然后卖到黑、灰色渠道。

其三，消费者是职业打假人，或者真货被身边的人掉包了（也不排除是付款人）。

这个案子的核心是品牌方、门店拿不出证据证明假货是否就是原来购买的那个。在这样的情况下，哪怕品牌方、仓储、门店、店员都没问题，也只能打碎牙往肚子里咽。

预测一下LV的公关策略：

上策，如果能收集到有力证据，积极提交给法院申请再审。胜诉之后反手上诉消费者敲诈勒索，要求消费者道歉并消除影响，全网发布澄清消息，完美结束。

中策，如果收集不到有力证据，或者查出品牌方、仓储、门店、店员有内鬼（就算没有也可以假定有）。全网发布调查报告，承认管理失误并承诺整改，对买到假货的消费者登门道歉获得谅解。

下策，虽然收集不到有力证据，但是一口咬定自己没有售假，不断上诉骚扰消费者。花重金雇佣水军全网洗地，设立受害者形象，煽动网络暴力。

我猜测LV用中策的可能性最大，如果真用了下策，那LV的公关部门可以祭天了。

# 2022年5月14日

学科培训企业转型困难的原因有很多，但总结一句话就是：拿不起放不下。拿不起是拿不起新产品和新业态，放不下是放不下原来的红利和安逸。

教培行业红利期太舒服了，好像随便摆几张桌椅就可以开课赚钱，导致大家都是野蛮生长，经营上怎么省事怎么来。教培行业产品结构单一，缺乏危机预案，面对“双减”和新冠的“围剿”自然招架不来。

你要让他们放弃以前的红利和安逸，再度进入水深火热，想必大多数人都不会愿意。实际上，当前的情况已经无异于重新创业了，与其修修补补又一年，还不如即刻调转航向，在沉船之前回港口换一艘新船。

前者看似稳妥但危机四伏，后者虽有难度但潜力更大。切勿“路径依赖”，切勿“路径依赖”，切勿“路径依赖”，重要的事情说三遍。

补充资料：问如何判断自己是否处于路径依赖中，又如何脱离依赖，且找到另一条路?

仔细回想你做事情的习惯，比如：遇到一个问题或者面临一项困难，你是先结合实际情况分析问题，还是习惯性从过去的经验里找答案。

举个例子：你一路上经过三条河，前两条河水很浅，趟着水就能轻松过去。到了第三条河，“路径依赖”的人就会想：“前两次水浅可以直接趟过去，这次我也这样过”，结果恰好这次水深就淹死了。

不“路径依赖”的人，会先观察水位或者用木棍之类的测量一下水位，再考虑如何过河。所以从某种程度上来说，经验也不一定可靠。

最简单的脱困方法是：遇到问题不要急着做选择，在做选择之前加上一个环节：分析实际情况。更不要有“我以前是这样做的，这次也这样做”的依赖心理，这样可以避免很多问题。

经验不一定是错的，有可能你分析之后还是发现经验里的做法最好，但是“习惯性”凭经验做事却有很大风险。因为“人不可能两次踏进同一条河流”，哪怕是同样的问题，在客观条件不同的情况下，以前的方法也不一定奏效。

# 2022年4月15日

看完杰克·特劳特的《定位》一书，里面很多案例都是：我提出了一个很好的方案，但是客户没有坚持下来，所以他们失败了。我觉得华与华在总结“超级符号”方法论时肯定参考过“定位论”，他们二者的核心很相似，区别也很明显。

特劳特的观点更抽象，像一位专家给公司献策。华与华的观点更接地气，从看得见摸得着的地方入手，比如设计、广告语。从市场角度看华与华更容易成交，但是从长远发展看，特劳特更利于企业常青。所以完全可以用特劳特的理论，华与华的方法，把二者结合一下。

把华与华的“超级符号”看做“定位论”本土化的成果也不是不可以，中国的企业家务实一点，不喜欢整意识形态那一套。照搬过来肯定是水土不服的，反而像华与华这样看起来“不伦不类”的更好，毕竟当年是以美国为基础提出的理论，兼容性上还是有待考究。

# 2022年4月10日

之前因为有部分容易读错的字词被主管部门“将错就错”，网上就冒出很多反对的言论。但是你仔细观察，就会发现这些人其实很多都自诩为“文化人”，把教条当“真理”，别人的观点不一样就是原罪。从应试和研究角度挑毛病无可厚非，但是文字是服务于大众交流的，又不是象牙塔里的雕塑，遥不可及。

文字的出现本来就是社会口语形成共识了，为了方便交流规范一下，做这样的改动也正是遵循了文字发展的规律，没有什么不妥的。普通人只要记住常用的读音就行了，毕竟不是专门做汉语言研究的，也不会记住“茴”有几种写法。

# 2022年3月14日

家庭教育现在最大的问题就是没有形成切实可行的商业模式和行业标准，大多数情况下还是赚一波是一波，不可持续。家长不可能像学龄期的小孩一样有足够的时间上课，但短期课程效果是个大问题。所以，家庭教育的未来业态能否像医疗行业的家庭健康顾问一样，是长期的、陪伴型的，能做孩子一生的导师和朋友？我认为这也许是最契合未来社会形态的一种方式。

好像绝大多数公司都害怕服务周期过长，害怕服务过程中出岔子。但是反过来想，服务周期越长，和客户的信任不是越牢固吗？就好像每个社区的诊所，大病你可能不会找他们，但是小病几乎是无条件信任，相互都很熟，没有太多的防备，有时候一些销售行为在客户眼里就成了朋友间的推荐，这不就是很多公司想要的销售氛围吗？或许真的可以一试。
